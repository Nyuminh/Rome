**Colosseum III** is an immersive action-RPG set in the heart of Ancient Rome. Step into the sand as a gladiator, master the art of the blade, and fight for glory, honor, and your ultimate freedom within the walls of the world's most legendary arena.

---

## üéÆ Key Features

* **Tactical Combat System:** Experience visceral melee combat featuring combos, parries, and directional dodging.
* **Dynamic AI Rivals:** Battle against diverse Roman legionaries and fierce beasts with advanced flocking and flanking behaviors.
* **Atmospheric Environment:** High-fidelity recreation of the Colosseum using URP, featuring realistic lighting and particle effects.
* **Progression System:** Earn gold from victories to upgrade your weaponry, shields, and gladiatorial armor.

## üõ†Ô∏è Tech Stack

* **Engine:** Unity 2021.3+ (Universal Render Pipeline - URP).
* **Language:** C# (Object-Oriented Design).
* **Version Control:** Git with **Git LFS** for high-resolution textures and 3D models.
* **Asset Pipeline:** Modeled in Blender, textured in Substance Painter.

## üöÄ Getting Started

To get a local copy up and running, follow these steps:

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/your-username/Colosseum-III.git](https://github.com/your-username/Colosseum-III.git)
    ```
2.  **LFS Initialization:** Ensure you have [Git LFS](https://git-lfs.github.com/) installed, then run:
    ```bash
    git lfs pull
    ```
3.  **Open in Unity:**
    * Open Unity Hub.
    * Click **Add** and select the project folder.
    * Ensure you are using the correct Editor version to avoid library conflicts.
